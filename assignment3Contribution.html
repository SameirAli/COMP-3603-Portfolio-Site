Contributions to the task-centered system walkthrough are shown below:


Task step description Does the user have the knowledge/ training to do this? Is it believable that they would do it? Are they motivated? Comments/suggested fixes
The user has to navigate the home page and find the view map feature Yes, a small amount of training may be required for the user to be able to do it efficiently The home page is designed to be intuitive. As such, the user would want to interact with the icons on screen. The names of each icon allows the user to know what they represent and be able to find the feature they wish to use. However, the ability of an individual varies greatly from person to person. As such, a very small percentage of users may require some form of training in order to efficiently use the application. A help/tutorial feature may be added, to help users who require training to effectively use the application.
Determine which is the best available parking spot. Yes, the app tells the user what the colors and percentages on the map mean. The user can look at the different car parks and see how full they are based on the color of the icon, as well as the percentage displayed. These metrics would guide the user in picking the
Selecting a parking location to view Yes, a small amount of training may be required depending on the individual’s technological competency. The pins on the map are a major focus when viewing the map. As such, it is expected that the user will tap these pins and be greeted with the option to view these parking locations. For the small number of users that would face difficulties using this feature, the help/tutorial feature would be of use to them.
Viewing and understanding the information of the parking location page Yes, once the user has navigated to this page, they will be able to view it. It is expected that the user will observe the information presented to them and understand it. The information is organized to be intuitive to the user. A map focused on the selected cark park is given, as well as directions on how to get to that car park.
Using the given directions to get to the desired car park. Yes, the directions are as simple as directions the user would get from another person. As such, it is expected that a large number of users will intuitively understand them. Some users may still have trouble, in this case the gps marker may help to alleviate those issues. The user would follow the instructions given to them by the app, in order to reach the car park. If the user is having trouble with following the directions, the gps marker will give them better context for the directions given. It is possible for the user to get lost due to improperly following the directions. However, this is an error on the user’s part and the app would not be able to do much to fix this.
Checking the menu of the application Yes, the bottom right floating menu button is common in a large number of other apps. As such the users will have previous training in using these buttons. If the user is unfamiliar with it then some training may be required. After the user has accomplished their task, they may be intrigued in the other functions of the application. As such they will use the floating menu button to see any additional features. For the users that don’t understand how to use the floating menu button. The help/tutorial feature will be of use to them
